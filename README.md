# 會員制電子商務購物系統 (PHP & MySQL)

本專案為一套完整的電子商務後台管理與前端購物系統，基於 PHP 與 MySQL 開發，旨在實作資料庫關聯設計與多重使用者權限控管。

## 系統核心功能

* **多重權限管理系統 (RBAC)：** 實作消費者、一般會員與管理員三種角色權限。
    * 管理員：具備產品增刪查改 (CRUD) 權限與後台管理介面。
    * 一般會員：可進行登入、個人資料管理及專屬功能操作。
* **產品管理系統：** 動態串接資料庫顯示產品資訊，支援管理端即時更新產品數據。
* **安全性認證：** 實作會員註冊、登入與登出流程，並包含密碼找回功能邏輯。
* **響應式設計：** 使用 CSS 進行介面美化，確保不同設備下的瀏覽體驗。

## 技術架構

* **後端開發：** PHP (實作業務邏輯與資料庫互動)
* **資料庫系統：** MySQL / MariaDB (實作多表關聯與資料持久化)
* **開發環境：** XAMPP (Apache + MariaDB)
* **前端技術：** HTML5, CSS3

## 檔案結構說明

* `*.php`: 核心系統邏輯與介面程式碼。
* `style.css`: 全站視覺樣式設定。
* `shop.sql` (或相關 SQL 檔): 完整的資料庫結構與預設數據腳本。
* `管理端/`: 專屬後台管理功能模組。

## 快速開始與展示

1.  **環境設定：** 使用 XAMPP 環境啟動 Apache 與 MySQL 服務。
2.  **資料庫匯入：** 透過 phpMyAdmin 建立資料庫，並匯入本專案附帶之 SQL 檔案。
3.  **運行系統：** 將程式碼放入 `htdocs` 資料夾後，透過瀏覽器存取 `index.php`。
